About Me 💡

1. Perkenalan 👋

Hallo! Saya Rafli, seorang pelajar yang sedang belajar dunia pemrograman dan desain web. Saya tertarik pada teknologi sejak kecil, dan sekarang mulai fokus belajar HTML, CSS, dan JavaScript.

2. Minat 💻

Saya ingin masuk jurusan PPLG karena saya ingin menjadi developer profesional. Saya suka pada web development, logika pemrograman, dan User Interface (UI).

3. Tujuan 🎯

Saya ingin masuk jurusan PPLG dikarenakan saya ingin mengembangkan minat ku dibidang web development agar bisa menjadi web development yang bisa membangun proyek yang baik dan benar

4. Hobby 🧠

Selain ngoding, saya juga senang bermain game dan belajar desain UI. Web ini saya buat sebagai salah satu portofolio dan bukti keseriusan saya untuk masuk ke dunia programming.

Skill

HTML (HyperText Markup Language) adalah bahasa yang digunakan untuk membangun struktur dasar sebuah halaman web. Saya memahami cara membuat halaman HTML dengan elemen-elemen seperti div, img, a, dan form input. HTML adalah fondasi penting sebelum mempelajari CSS dan JavaScript.

CSS membantu membuat tampilan web menjadi lebih menarik,saya sudah belajar cara mengubah warna, mengatur jarak antar elemen, menggunakan flexbox, dsb. namun saya sadar banyak hal yang belum saya kuasai seperti responsive design, atau layout yang kompleks. Oleh karena itu saya terus belajar sambil mencoba langsung di web kali ini
